Download (Far1/Far2 x86/x64)

About

Fast and easy editor of windows associations Far Manager plugin.

How to use?

Install plugin first, then F11 FileTypes, or use plugin prefix - type in command line: 

ftypes:

or 

ftypes:<extension>

and press Enter.
